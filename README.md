# 🎨 Random Color Generator

A beginner-friendly JavaScript project that generates a random RGB color each time the button is clicked. The generated color is displayed as text and applied as the background color of the preview box.

## ✨ Features

- Generate Random RGB Colors
- Dynamic Background Color Update
- DOM Manipulation
- Event Handling
- Clean and Responsive UI
- Bootstrap Styled Button

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

---

## 📚 Concepts Practiced

- DOM Selection
- Event Listeners
- Functions
- Math.random()
- Math.floor()
- Template Literals
- CSS Manipulation using JavaScript
- Dynamic Content Update

## 🚀 How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Click **Generate Color** to create a new random RGB color.
